# Política de Privacidade - OrganizaDin

**Última atualização:** 28 de Janeiro de 2026

## 1. Introdução

O OrganizaDin ("nós", "nosso" ou "app") respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política de privacidade descreve como coletamos, usamos e protegemos suas informações.

## 2. Dados que Coletamos

### 2.1 Dados Armazenados Localmente

O OrganizaDin é um aplicativo **offline-first** que armazena todos os seus dados financeiros **exclusivamente no seu dispositivo**:

- Transações financeiras (receitas e despesas)
- Cartões de crédito e suas informações
- Categorias personalizadas
- Metas de economia (porquinhos)
- Configurações do aplicativo
- Fotos de compras (armazenadas localmente)

**Importante:** Seus dados financeiros NÃO são enviados para servidores externos. Tudo permanece no seu dispositivo.

### 2.2 Dados de Uso

O app pode coletar dados anônimos de uso para melhorar a experiência:
- Informações sobre crashes e erros técnicos
- Estatísticas de uso de funcionalidades

Estes dados são **anônimos** e não contêm informações financeiras pessoais.

## 3. Como Usamos Seus Dados

Os dados são usados exclusivamente para:
- Fornecer funcionalidades do aplicativo
- Permitir que você gerencie suas finanças pessoais
- Melhorar a experiência do usuário
- Corrigir bugs e problemas técnicos

## 4. Segurança dos Dados

Implementamos diversas medidas de segurança:

### 4.1 Armazenamento Seguro
- Dados sensíveis armazenados com **Expo SecureStore**
- Criptografia nativa do sistema operacional (iOS Keychain / Android Keystore)
- Senha de 4 dígitos para acesso ao app (opcional)

### 4.2 Proteção de Banco de Dados
- Validação de entrada contra SQL Injection
- Sanitização de dados
- Backup/restore com validação de integridade

### 4.3 Proteção de Tela
- Opção de bloquear capturas de tela em telas sensíveis
- Ocultação de conteúdo ao alternar entre apps

### 4.4 Proteção contra Root/Jailbreak
- Detecção de dispositivos comprometidos
- Alertas de segurança quando apropriado

## 5. Compartilhamento de Dados

**NÃO compartilhamos, vendemos ou transferimos seus dados pessoais para terceiros.**

O OrganizaDin não possui servidores próprios e não transmite suas informações financeiras para nenhum servidor externo.

## 6. Backup e Exportação

Você pode:
- Criar backups locais dos seus dados
- Exportar dados em formato JSON
- Restaurar dados de backups anteriores

**Importante:** Você é responsável por manter seus backups em local seguro. Recomendamos:
- Não compartilhar arquivos de backup
- Armazenar em local seguro (nuvem privada, local criptografado)
- Não enviar backups por e-mail ou mensagens não criptografadas

## 7. Permissões do Aplicativo

### 7.1 Android
- **Câmera:** Para tirar fotos de compras
- **Galeria/Fotos:** Para anexar fotos existentes às compras
- **Armazenamento:** Para salvar/carregar backups locais

### 7.2 iOS
- **Câmera:** Para tirar fotos de compras
- **Biblioteca de Fotos:** Para anexar fotos existentes às compras

**Nota:** Todas as permissões são solicitadas apenas quando necessárias e podem ser revogadas nas configurações do dispositivo.

## 8. Direitos do Usuário

Você tem direito a:
- Acessar todos os seus dados (estão no seu dispositivo)
- Exportar seus dados via funcionalidade de backup
- Apagar todos os dados (desinstalar o app ou usar função de reset)
- Solicitar informações sobre como seus dados são processados

## 9. Retenção de Dados

Os dados permanecem no seu dispositivo até que você:
- Desinstale o aplicativo
- Execute a função de reset/limpar dados
- Apague manualmente os dados

## 10. Crianças

O OrganizaDin não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

## 11. Alterações nesta Política

Podemos atualizar esta política periodicamente. Alterações significativas serão notificadas através de:
- Atualização da data no topo deste documento
- Notificação no app (quando aplicável)

## 12. Conformidade Legal

Este aplicativo está em conformidade com:
- LGPD (Lei Geral de Proteção de Dados - Brasil)
- GDPR (General Data Protection Regulation - União Europeia)
- CCPA (California Consumer Privacy Act - EUA)

## 13. Código Aberto e Transparência

O OrganizaDin é desenvolvido com foco em transparência:
- Código auditável
- Sem dependências suspeitas
- Sem rastreadores ou analytics invasivos
